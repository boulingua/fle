Site FLE par **S. Le Boulanger** pour l'enseignement secondaire,
classes 6 à 13. Tout le contenu — unités, exemples d'épreuves,
fiches — est sous **CC-BY-SA 4.0** (`LICENSE-content`). Le code
est sous **MIT** (`LICENSE`).

## Pourquoi ce site

Le français comme deuxième langue vivante est sous-représenté
dans le matériel ouvert FLE. Ce site veut combler le manque : un
curriculum cohérent, aligné sur un référentiel
(« Bildungsplan ») régional, qui se laisse réutiliser, remixer,
étendre.

## Ce que ce site n'est pas

- **Aucune paraphrase de manuels** Hachette FLE, CLE
  International, Didier, Klett. Tous les textes sont originaux.
- **Aucun texte verbatim d'épreuves passées.** Les exemples
  d'épreuve s'orientent au format officiel, mais avec des
  stimuli originaux.
- **Aucune image sous droit d'auteur.** Icônes Lucide (ISC),
  SVG originaux uniquement.
